🔒 ChatBot 난독화 버전 테스트

1. 원본 버전 (Original) 보안: 낮음

• 읽기 쉬운 코드로 디버깅과 개발에 적합

• 변수명과 함수명이 명확하게 표시됨

• 주석과 공백 포함

파일 크기: ~22KB (788 줄)
2. 압축 버전 (Minified) 보안: 보통

• 공백과 주석 제거로 파일 크기 최적화

• 기본적인 변수명 단축

• 네트워크 전송 최적화

파일 크기: ~15KB (6 줄)
3. 기본 난독화 (Basic Obfuscated) 보안: 높음

• 변수명과 함수명 암호화

• 문자열 배열화 및 인코딩

• 코드 흐름 복잡화

• 기본적인 역공학 방지

파일 크기: ~37KB (1 줄)
4. 고급 난독화 (Advanced Secure) 보안: 최고

• Self-Defending: 개발자 도구 감지 및 차단

• Debug Protection: 디버깅 방지

• Control Flow Flattening: 코드 흐름 완전 복잡화

• Dead Code Injection: 가짜 코드 삽입

• String Array Encoding: Base64 문자열 인코딩

• Unicode Escape: 유니코드 이스케이프 시퀀스

파일 크기: ~168KB (1 줄)

🧪 테스트 방법

  1. 위 버튼 중 하나를 클릭하여 해당 버전을 로드하세요
  2. 오른쪽 하단에 채팅 버튼이 나타나는지 확인하세요
  3. 채팅봇 기능이 정상 작동하는지 테스트하세요
  4. 개발자 도구(F12)를 열어서 코드를 확인해보세요
  5. 고급 난독화 버전에서는 개발자 도구가 차단될 수 있습니다

⚠️ 주의사항

  • 고급 난독화 버전은 개발자 도구를 감지하고 차단합니다
  • 디버깅이 필요한 경우 원본 또는 압축 버전을 사용하세요
  • 프로덕션 환경에서는 보안 수준에 맞는 버전을 선택하세요
  • 파일 크기와 보안 수준은 트레이드오프 관계입니다
