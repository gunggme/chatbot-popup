🤖 ChatBot Popup Library

채널톡 스타일의 현대적인 채팅봇 팝업 라이브러리

✨ 실시간 데모

오른쪽 하단의 채팅 버튼을 클릭하여 채팅봇을 체험해보세요!

🚀 빠른 시작

단 한 줄의 코드로 바로 시작하세요:

<script src="https://cdn.jsdelivr.net/npm/chatbot-popup-library@1.0.0/lib/chatbot-library.min.js"></script>

⚙️ 커스텀 설정

<script> const chatbot = new ChatBot({ apiBaseUrl: 'https://your-api-server.com/api/chat', profileId: 'your-profile-id', theme: { primaryColor: '#007bff', secondaryColor: '#6c757d', backgroundColor: '#ffffff' } }); </script>
